## KEYWORDS SEO PARA CADA PRODUCTO

Al generar el producto, incluir estas variaciones:
- extintor + [tipo] + [capacidad]
- extintor + para + [sector]
- extintor + [tipo] + CDMX
- extintor + certificado + NOM-154
- venta de extintor + [tipo]
- comprar extintor + [capacidad]
- extintor contra incendios + [aplicacion]

### Ejemplos:
- "extintor PQS 4.5 kg"
- "extintor para restaurante CDMX"
- "extintor CO2 certificado NOM-154"
- "venta de extintor tipo K"
- "comprar extintor 6 kg"
- "extintor contra incendios oficina"

## NOTAS IMPORTANTES

1. **Certificacion**: Todos los productos deben mencionar NOM-154-SCFI
2. **Clases de fuego**: Verificar las clases correctas por tipo
3. **Sectores**: Mencionar aplicaciones especificas en la descripcion
4. **SEO**: Incluir keywords de busqueda local (CDMX, Mexico)
5. **CTAs**: Incluir llamados a accion para cotizacion
